# Sticky for Claude

**Claude, floating above everything on your Mac, even true fullscreen.**

Sticky for Claude is a native macOS app that puts Claude in floating sticky-note windows that stay above every app, including true fullscreen (the same window layer Spotlight uses, no screen-recording permission). Each window can be a Claude chat, a full Claude Code session pinned to a project folder (slash commands, tools, permission modes), or a real terminal running the Claude Code CLI.

## Highlights

- **Above everything**: windows survive true fullscreen, on every Space
- **Unlimited windows**: one per project, task or topic, side by side
- **Claude Code, everywhere**: chat mode, full Claude Code pinned per project, or a real floating terminal
- **Out of your way**: windows auto-fade when idle; double-tap ⌘ summons or hides them anywhere
- **Private by design**: no account, chat history stays in local files on your Mac
